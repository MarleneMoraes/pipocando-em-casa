Sobre

Pipocando em Casa é um website brasileiro que trazem notícias sobre filmes, seja no cinema, serviços de streaming e premiações. O site está no ar desde 04 de outubro de 2020. Este website foi desenvolvido com o intuito de avaliação para a disciplina Desenvolvimento de Interfaces Web, da PUC Minas São Gabriel.

O site obteve uma segunda versão em 05 de dezembro de 2021, após várias tentativas frustradas de refatorá-lo, com os conhecimentos mais aprofundados em front-end, sobretudo a ferramenta Bootstrap 4.
